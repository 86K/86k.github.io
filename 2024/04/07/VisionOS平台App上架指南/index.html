visionOS平台App上架指南

itsxwz
2024-04-07
App Store, visionOS

一、官方网址

官方地址(总提交页)
使用App Store Connect进行提交App

二、提交流程

App Store Connect -> App -> +

注册套装ID

  • Description：应用程序名称
  • Bundle：com.company.product
    注：注册一个app ID之后，则可以在新建App页面的“套装ID”项中进行选取。

SKU（Stock Keeping Unit）

App ID Prefix + Bundle ID

1.0准备提交

准备预览图和截屏（3840 x 2160px）

  • 注：模拟器截图的是2732x2048，使用ps进行缩放。
  • 填写必要信息

xcode分发

  • 必须选择一个构建版本
  • 必须要定价中选择价格等级
  • 必须在“App 隐私”页面输入隐私政策网址。不搜集任何数据，则不用填写隐私政策网址。
  • 必须在App信息中设置内容版权信息
  • … 这些在点击“添加以供审核”按钮时，会提示
    注：没有真机，不能用xcode直接分发到app store。会报错：Invalid Run Destination。

distribute情况

  • Any visionOS Simulator Device(arm64)不可以分发
  • 采用分发到iOS平台的方式：Any iOS Device(arm64) + Supported Destination增加iPhone和iPad
  • 必须先build一次再分发，但是archive不可distribute
  • archive页面显示Unity-VisionOS(Other Item)的archive项，distribute app和validate app按钮灰色不可点击，Type为Generic Xcode Archive
