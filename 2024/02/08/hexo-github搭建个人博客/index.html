hexo+github搭建个人博客

itsxwz
2024-02-08
tutorial

一、环境准备

  • 安装npm
  • 安装hexo

二、博客准备

hexo官网 -> https://hexo.io/index.html

  • ##全局安装hexo
  • npm install hexo-cli -g
  • ##hexo初始化
  • hexo init blog
  • ##跳转到blog目录
  • cd blog
  • ##安装npm包
  • npm install
  • ##把下载的主题放到themes文件夹下
  • 主题官网：https://github.com/probberechts/hexo-theme-cactus
  • ##修改_config.yml中theme:xxx，xxx为主题文件夹名字
  • ##创建一篇新博文
  • hexo new “博文名字”
  • ##在博客目录下生成静态页面
  • hexo g
  • ##启动博客的本地服务器
  • hexo s

三、发布到github

  • ##仓库settings下设置github pages
  • branch和domain设置（一般仓库名xxx.github.io，xxx与github账户名一致）
  • ##deploy下修改
  • type: git
  • repo: 仓库地址
  • branch: master
  • ##安装hexo-deployer-git
  • npm install hexo-deployer-git –save
  • ##部署（可能需要配置邮箱和名字）
  • hexo d
